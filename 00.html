1- ¿Cuáles son los tipos de títulos que hay?

Los tipos de textos contienen 6 niveles, yendo desde el nivel h1 al h6, h1 es el más importante, y h6 el menos importante

Estos títulos describen brevemente el tema de la sección que presenta.La información de los títulos puede ser usada por los agentes usuarios, por ejemplo, para construir una tabla de contenidos para un documento automáticamente.

5- ¿Qué es un atributo en una etiqueta HTML? Dar dos ejemplos de etiquetas HTML con atributos, marcando sus respectivos atributos.

Un atributo en una etiqueta HTML describe las características de la etiqueta contienen información extra sobre el elemento que no se mostrará en el contenido.

donde "class" es el atributo: p class="editor note"> my cat is so silly", donde este es una lista de clases separadas por espacios, implícito e identificador a nivel de documento

donde "style" es el atributo: p style="editor note">"abc", donde esto son las reglas de estilo, implícito e información de estilo en línea.

6- ¿Qué atributos tienen las etiquetas de título?. Definir.

Los atributos globales son atributos comunes a todos los elementos HTML; se pueden usar en todos los elementos, aunque es posible que no tengan ningún efecto en algunos elementos.

Los atributos globales se pueden especificar en todos los elementos HTML, incluso aquellos no especificados en el estándar.Eso significa que cualquier elemento no estándar debe permitir estos atributos

8- Investiga qué hacen y cómo se utilizan las siguientes etiquetas HTML

a) strong: El elemento strong es el apropiado para marcar con especial énfasis las partes más importantes de un texto.

b) i: Muestra el texto marcado con un estilo en cursiva o italica.

c) u: Muestra el texto subrayado.

d) sub: El elemento HTML sub define un fragmento de texto que se debe mostrar, por razones tipográficas, más bajo, y generalmente más pequeño, que el tramo principal del texto, es decir, en subíndice.

e) sup: El elemento HTML sup define un fragmento de texto que se debe mostrar, por razones tipográficas, más alto, y generalmente más pequeño, que el tramo principal del texto, es decir, en superíndice.

f) em: El elemento HTML em es el apropiado para marcar con énfasis las partes importantes de un texto. El elemento em puede ser anidado, con cada nivel de anidamiento indicando un mayor grado de énfasis.

g) a: El Elemento HTML Anchor a crea un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma página, direcciones de correo, o cualquier otra URL.

h) span: span - abarcar. Es un contenedor en línea. Sirve para aplicar estilo al texto o agrupar elementos en línea.

9- ¿Cuál es la estructura básica del HTML?

La estructura basica del html es:

!doctype html: es el tipo de documento. Es un preámbulo requerido.

html: Este elemento encierra todo el contenido de la página entera y, a veces, se le conoce como el elemento raíz (root element).

head: Este elemento actúa como un contenedor de todo aquello que quieres incluir en la página HTML que no es contenido visible por los visitantes de la página.

meta charset="utf-8" /: Este elemento establece el juego de caracteres que tu documento usará en utf-8, que incluye casi todos los caracteres de todos los idiomas humanos. Básicamente, puede manejar cualquier contenido de texto que puedas incluir.

title Mi pagina de prueba /title: establece el título de tu página, que es el título que aparece en la pestaña o en la barra de título del navegador cuando la página es cargada, y se usa para describir la página cuando es añadida a los marcadores o como favorita.

/head

body: Encierra todo el contenido que deseas mostrar a los usuarios web que visiten tu página, ya sea texto, imágenes, videos, juegos, pistas de audio reproducibles, y demás.

img src="images/firefox-icon.png" alt="Mi imagen de prueba" /: incrusta una imagen en la página, en la posición en que aparece. Lo logra a través del atributo src

/body

/html
